# Changelog

## 0.1.0 - 2026-06-07

- Initial open source project structure.
- Added JSONL and SQLite embedding cache loaders.
- Added audit rules for dimensions, duplicates, empty vectors, non-finite values, model mixing, missing ids, missing chunks, and required metadata.
- Added JSON and Markdown reporters.
- Added CLI commands for `inspect jsonl`, `inspect sqlite`, and `schema options`.
- Added examples and standard-library test suite.
